## 사용 방법

#### start

```
npm install
npm vite preview
```

#### build

```
npm run build
```

#### dev

```
npm run dev
```

## 🧸 사용 기술 이슈 
- Vite를 이용하여 환경변수 구축 및 배포, 빌드 진행
- express를 이용하여 백엔드 구축 및 통신 기능 개발
- cloud flare를 활용한 사이트 배포 진행
- react-query를 이용하여 openAI API와 통신 구축
